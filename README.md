# Product Showcase Site

Базовый сайт-витрина для твоих расширений и приложений.

## Что есть

- Главная страница с карточками продуктов и фильтром по сервисам `Litnet`, `Litmarket`, `Litgorod`
- Регистрация первого администратора по email и паролю
- Вход в админ-панель через сессию
- Загрузка изображения и ZIP-архива для нового продукта
- Хранение данных без базы: `data/*.json` и локальные файлы в `uploads/`

## Запуск

```bash
npm install
npm start
```

После запуска сайт будет доступен на `http://localhost:3000`.

## Полезно знать

- Первый администратор создаётся через страницу `/register`
- После появления первого админа открывается обычный вход через `/login`
- Для продакшена задай переменную окружения `SESSION_SECRET`
